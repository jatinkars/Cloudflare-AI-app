# Cloudflare-AI-app
cf_ai_edge_chat_memory is a lightweight AI chat app on Cloudflare Workers. It uses Workers AI (Llama 3.3) and a Durable Object to keep per-session memory, with a simple web chat served by the same worker. One repo with a clear README and PROMPTS.md. Easy to run with Wrangler and deploy to Cloudflare.
